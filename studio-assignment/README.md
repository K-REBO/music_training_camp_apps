# Webベースの合宿用スタジオ並び替えアプリ
## 要件
- 登録された人名 or 登録されたバンド名から予約する
- 部屋数・時間枠数は設定可能
- URLにアクセスで書き込み
- モバイル向けCSS

## FLOW
1. band.jsonを読み込み、バンドと設定をロードする
2. 以下の条件にできるだけ従うようなアルゴリズムで時間割形式に並び替えを行う
	- 条件
	1. 横の並び(部屋の並び)に同じ人間が含まれないようにする。(同時に練習に参加できないため)
	2. Key,Gt,Ba,Drの順に優先し、縦の並び(時間の並び)で同じ部屋になるように、配置する。(この順で移動が面倒なため)
3. 並び替えられた時間割をJSONに書き出す。

## レイアウト
横軸：練習部屋 (Room1, Room2, ...)
縦軸：時間枠 (1時間単位想定)

## 構成
- Svelte/TailwindCSS
- pnpm
